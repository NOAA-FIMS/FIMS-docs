◆ finalize_fims()

void finalize_fims ( Rcpp::Nullable< Rcpp::List >  obj = R_NilValue,
Rcpp::Nullable< Rcpp::List >  opt = R_NilValue 
)

Finalizes a FIMS model by updating the parameter set. This function evaluates the objective function and the gradient with the given parameter set.

Parameters
obj- Either a list containing "fn" and "gr", or a list containing two seperate lists "obj" and "opt", "obj" should contain "fn" and "gr", "opt" should contain "par". In the second case, the second function argument is expected to be null and ignored.
opt- A list containing "par".
